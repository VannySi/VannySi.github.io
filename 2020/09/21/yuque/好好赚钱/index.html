好好赚钱

发表于 2020-09-21 更新于 2020-10-04

用时 6 时 28 分

真正的安定是拥有对于金钱的把控力。

关于本书

  1. 结构不好，有卖课嫌疑，不建议阅读
  2. 笔记结构有调整，与书不一致

认识投资

思维升级

  1. 好好工作 != 好好赚钱
  2. 理财 != 节约、省钱
  3. 财富反面
    1. 意外 => 应急资金
    2. 疾病 => 保险
    3. 无规划支出 => 量入为出，消费降级
    4. 脆弱的投资体系 => 多读书多思考多总结
  4. 富人思维
    1. 跨越稀缺陷阱
    2. 注重长期回报
    3. 由目标找资源
    4. 不给自己设限
  5. 理财三要素
    1. 本金累积
    2. 利率提升
    3. 时间累积
  6. 杠杆自由：当借款成本合理时应该尽可能的使用杠杆。

上手实践

  1. 方法论：1）任务分解 2）设立补给站 3）分清目标和愿望

财务状况

理财规划

  1. 设定目标（SMART 原则）
  2. 把「钱」点清
  3. 财务「体检」
  4. 测算未来
  5. 行动指南

明智购物(消费降级)

  1. 搬家炸裂法(断舍离)
  2. 10% 「品质生活」基金
  3. 买资产不买负债
  4. 多为自己买单

告别月光(量入为出)

  1. 理解财富水池
  2. 优化收支结构
  3. 消灭拿铁因子
  4. 培养储蓄习惯：i. 先支付自己 ii. 10/50 法则(收入存 10%，意外(奖金等)存 50%)
  5. 管理财富水池

检查财务状况

  1. 应急能力 => 应急资金 = 每月支出 * 3 ～ 6 月 => 货基、活期、定期(12 单法)
  2. 偿债能力 => 月还款金额/月收入 < 1/3(杠杆自由)
  3. 保障能力 => 保险
  4. 生息能力 => 可投资资产/总资产 >= 30%，总资产收益率 >= 5%。

借钱

  1. 为好东西借钱(房子、能力提升)
  2. 有偿还能力
    • 总负债/总财富 <= 40%
    • 月还款/月收入 < 1/3
  3. 成本低
  4. 渠道：亲戚朋友、银行(消费贷)、信用卡取现、现金贷、金融平台(借呗、微粒贷、京东金条)

还款

  1. 收入扣除还款金额后再做日常支出计划
  2. 将还款日调整为工资日后一天
  3. 购买月收款理财，月收款大致等于还款金额

信用卡

保险：五险一金 + 其它

投资方法

确认目标(年化收益率)

  • 预期收益率 = 无风险收益率 + 风险溢价
  • 保守：货基、银行理财、券商理财、保险理财，5%左右
  • 稳健：保守 + 网贷投资、权益类资产，5%～ 8%
  • 积极：权益类，10%+
  • 根据资产配置，年化收益率在 6%～ 12% 是风险/收益的性价比最好(个人观点)

实现目标

投资分类：现金类、债权类、权益类、商品实物类

  1. 活期投资
    1. 货基
    2. 活期理财
  2. 银行存款
    1. 货比三家
    2. 大额存单(20W+)：提前支取&直接转让
    3. 通知存款(5W+)：收益计算&部分支取
  3. 国债
    1. 电子储蓄式：每年付息，到期还本
    2. 凭证式：到期一次性还本付息
    3. 记账式：利息 + 买卖差价
    4. 国债 ETF：分红 + 买卖差价
  4. 理财产品
    1. 银行发行
    2. 证券公司发行
    3. 保险公司发行(分红险、万能险、投连险)
  5. 基金
    1. 基金类型
      • 货币基金：国债、央行票据、商业票据、银行定存、同业存款等
      • 债券基金：债券(80%+) + 其它
      • 股票基金：股票(80%+) + 其它
      • 混合基金
    2. 投资心态
    3. 费用
      1. 银行：波动性越大费用越高；申赎费无优惠
      2. 基金公司：申赎费打折，基金间可转换；选择性少
      3. 第三方基金代销平台：手续费低，品种齐全
    4. 指数基金
  6. 投资组合(极简投资法)

管理财务

其它

  • 财务自由度 = 投资净收益 / 总支出，建议 >= 15%。
  • 财务自由三阶段：觉醒(认识投资)，养鹅(财务状况)，建立自己的投资系统(投资方法)
  • SMART 原则
  • 好物清单(有删减)
    • 正版书籍
    • 好化妆护肤品
    • 适合得体衣服
    • 好的床上用品
    • 定期保洁服务
    • 视频网站会员
    • 高质量的旅行
    • 私人健身课程
    • 付费培训课程
    • 好生产力工具
    • 优秀付费软件
